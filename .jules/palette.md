## 2026-05-11 - Clearing Focus on Hide
**Learning:** In WoW addons, `EditBox` components can trap user keyboard input. Always explicitly clear focus (e.g., calling `element:ClearFocus()`) on `OnEnterPressed`, `OnEscapePressed`, and when the parent frame is hidden (`OnHide`), to prevent locking out standard game keybinds. This is specifically relevant to the `OnHide` event, which was not consistently configured for `EditBox`es in the UI.
**Action:** Append an `OnHide` event handler (using `HookScript`) to all `EditBox` fields in `UI/Common.lua` that clears focus, ensuring that standard game keybinds are re-enabled when their parent frame is hidden.
